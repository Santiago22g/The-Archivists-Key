# 🗝️ The Archivist's Key

### 📄 Project Overview
**The Archivist's Key** is an atmospheric, story-driven escape room built for the web. Players are trapped within a sprawling, infinite library composed of the world's forgotten knowledge. To escape, you must act as a digital detective: analyzing classic texts, inspecting historical artifacts, and solving puzzles using real-world heritage. It is a playable tribute to the beauty of the Public Domain.

### Design Pillars
* **Literary Mystery:** Narrative-driven puzzles where the solution is hidden within the prose of history’s greatest authors.
* **Historical Immersion:** Every object, painting, and sound is a real piece of history sourced from global digital archives.
* **Minimalist Logic:** A focus on observation and "Aha!" moments rather than complex mechanics or combat.

### 🎨 Key Features
* **🧩 Interactive Curiosities:** Click and inspect high-resolution artifacts sourced from **The Public Domain Review**.
* **📖 The Gutenberg Cipher:** Use excerpts from **Project Gutenberg** as manuals to unlock mechanical safes and hidden compartments.
* **✨ Visual Restoration:** Solve puzzles to transform damaged images from **Europeana** into vibrant, restored masterpieces.
* **🎵 ASMR Feedback:** High-quality sound design featuring paper rustling, wooden clicks, and ambient library sounds.

### 🕹️ Game Structure
The game is organized into three main scenes to facilitate collaborative development and avoid conflicts in the engine.

| Phase | Scene | Description |
| :--- | :--- | :--- |
| **Intro** | Main_Menu.tscn | This will be the first room you must escape, from, it will work as a tutorial for the beginners. |
| **Core** | The_Chamber.tscn | The second Escape Room environment. Includes inventory logic and puzzles. |
| **Outro** | Last.tscn | Last screen and a detailed bibliography of the public domain sources used. |

### 🎮 Controls
Optimized for intuitive mouse interaction.
* **Interact / Pick Up:** Left Click
* **Examine / Zoom:** Right Click
* **Inventory:** Hover over the bottom of the screen
* **Pause / Menu:** Esc

### 🛠️ Tech Specs
* **Engine:** Godot Engine (Web/HTML5 Export).
* **Visuals:**
    * **Backgrounds:** 19th-century architectural photography from **Europeana**.
    * **Items:** Transparent PNG cutouts of historical engravings from **The Public Domain Review**.
* **Audio:**
    * **Music:** Period-accurate classical soundtrack (Satie, Debussy, or Chopin).
    * **SFX:** Physical foley (books, keys, and stone) to enhance immersion.
